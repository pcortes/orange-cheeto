## Short Description (132 characters max)

## Detailed Description (4,400 characters max)

Make the news bearable again.

Orange Cheeto is a fun Chrome extension that automatically replaces "Trump" with delightful nicknames everywhere you browse. Whether you're reading the news, scrolling social media, or just surfing the web, watch as "Donald Trump" transforms into "Orange Cheeto," "Mango Mussolini," "Cheeto Benito," and more.

**FEATURES**

- Automatic text replacement across all websites
- 7 built-in nicknames to choose from
- Add your own custom nicknames
- 4 playful animation styles (shimmer, glow, pulse, or none)
- Preserves original text capitalization (TRUMP → ORANGE CHEETO)
- Hover over any replacement to see the original word
- Badge counter shows replacements on current page
- Dark mode support
- Respects reduced motion preferences
- No tracking, no ads, no data collection

**BUILT-IN NICKNAMES**

- Orange Cheeto (enabled by default)
- Mango Mussolini (enabled by default)
- Cheeto Benito (enabled by default)
- The Tangerine Tyrant (enabled by default)
- Agent Orange (enabled by default)
- Dorito Mussolini (opt-in)
- Cheeto Jesus (opt-in)

Plus, add unlimited custom nicknames through the settings popup!

**ANIMATION STYLES**

Choose your vibe:
- Shimmer: A subtle light sweep effect
- Glow: Pulsing orange glow around text
- Pulse: Gentle scale animation
- None: Clean, static display

**HOW IT WORKS**

1. Install the extension
2. Visit any news site or webpage mentioning Trump
3. Watch the magic happen
4. Click the extension icon to customize settings
5. Share screenshots with friends

**PRIVACY FIRST**

We believe in privacy. This extension:
- Stores settings locally in your browser only
- Makes zero external network requests
- Collects zero data about you
- Contains zero tracking or analytics
- Is fully open source

**WHY?**

Sometimes you just need a little levity in your news consumption. This extension is political satire meant to bring a smile to your face during difficult news cycles. It's the digital equivalent of laughing so you don't cry.

## How to Take Screenshots

### Option 1: Developer Tools Device Emulation
1. Open Chrome DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. Set custom dimensions: 1280x800
4. Navigate to a news site
5. Take screenshot (Ctrl+Shift+P → "Capture screenshot")

### Option 2: Manual Browser Resize
1. Install "Window Resizer" extension temporarily
2. Set browser to 1280x800
3. Use system screenshot (Cmd+Shift+4 on Mac, Win+Shift+S on Windows)

### Option 3: Automated Screenshot Script
Use Puppeteer (see screenshot-generator.js in this repo)
